# Project-Solana

Description
Vue.Js project developed with solana/web3.js to show stats from Solana Blockchain

Usability
By accessing the web application the user will have access to charts with informations generated with Solana Blockchain data

How it works
The web application will gather information through the solana/web3.js and generate charts that will be displayed to the user.

Shown Information
Number of daily transactions
Number of daily active addresses
Number of daily new addressess
Daily volume of SOL transfers


# Project-Solana

Descrição
Projeto em Vue.Js desenvolvido com solana/web3.js para exibir informações da Blockchain Solana.

Usabilidade
Ao acessar a plicação web, o usuário terá acesso a gráficos com informações geradas a partir dos dados coletados na Blockchain Solana.

Como funciona
A aplicação web fará a coleta de dados através da API solana/web3.js e irá gerar gráficos com informações tratadas a serem exibidas ao usuário.

Informações a serem exibidas
Número de transações diárias
Número diário de endereços ativos
Número diário de novos endereços
Volume diário de transferências SOL
